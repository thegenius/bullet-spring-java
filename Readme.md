## Startup

### install tools
```text
1. install rust and cargo 
2. cargo install cargo-bullet 
3. cargo bullet install --name=bullet-spring-java 
```

### generate project 
```text
1. cargo bullet create --name=bullet-spring-java
2. cargo bullet build --name=bullet-spring-java
```
